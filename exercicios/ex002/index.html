Os Parágrafos

voce pode escrever um Parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>

Se precisar quebrar o texto em algum lugar específico
como esse voce pode usar a tag <br>

Para utilizar emoji podemos acessar o site emojipedia.org e escolher
na parte de CODEPOINTS um código do emoji desejado.

Lembre de trocar o 'U+' por '&#x'(é um exadecimal) para o código funcionar. ✅
